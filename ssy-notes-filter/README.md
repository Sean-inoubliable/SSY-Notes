## SpringBoot 中的 Filter

Filter 是 JavaEE 中 Servlet 规范的一个组件。

位于包 javax.servlet 中。

它可以在 HTTP 请求到达 Servlet 之前，被一个或多个 Filter 处理。

### 什么是过滤器
> 过滤器是对数据进行过滤、预处理过程。

当浏览器发送请求到服务器的时候，先执行过滤器，然后才访问Web资源。

服务器响应Response，从Web资源抵达浏览器之前，也会途径过滤器。

可以将其比作是一张 ‘滤网’。

### 什么时机会用到过滤器
- 当需要限制用户访问某些资源时
- 在处理请求时需要提前处理某些资源时
- 当服务器响应的内容需要进行处理再返回时

### 实现
- 只要Java类实现了Filter接口就可以成为过滤器。
